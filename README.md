# small_project_LP1
A small project for college

 - Copie o projeto para seu computador com o comando:
		`git clone https://github.com/FernandoIgorrr/SGBDinC.git`
 - Complile-o:
    `make`
 - Execute o binário:
    `./main`
 - Limpe os arquivos objetos (isso também vai apagar o binário, depois eu concerto isso):
    `make cls`
- Veja a documentação gerada pelo doxygen em HTML e LaTeX (você vai prescisar tero o doxygen instalado):
    `doxygen Doxyfile`
